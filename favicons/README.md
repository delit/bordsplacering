# Favicons

Runtbordsikon från startsidan (guld `#b8874d`, bordsyta `#f5ede0`, bakgrund `#faf7f2`).

| Fil | Storlek |
|-----|---------|
| `icon.svg` | Vektor (källa) |
| `favicon.ico` | 16, 32, 48 |
| `favicon-16x16.png` | 16×16 |
| `favicon-32x32.png` | 32×32 |
| `favicon-48x48.png` | 48×48 |
| `apple-touch-icon.png` | 180×180 |
| `icon-192.png` | 192×192 |
| `icon-512.png` | 512×512 |

Generera om PNG/ICO efter ändring i `icon.svg`:

```bash
npm install
node generate.mjs
```
